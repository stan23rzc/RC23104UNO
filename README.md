# RC23104UNO

Libreria Python para resolver sistemas de ecuaciones lineales y no lineales.

## Metodos implementados

### Sistemas lineales
- Eliminacion de Gauss
- Gauss-Jordan
- Regla de Crammer
- Descomposicion LU
- Jacobi
- Gauss-Seidel

### Sistemas no lineales
- Biseccion
